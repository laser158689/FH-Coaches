# Code of Conduct

## Purpose

This repository discusses field hockey coaching, recruiting, player development, mental skills, rules, and media analysis. Because those topics can involve minors, injuries, and personal performance details, community participation here requires stronger privacy and safety discipline than a generic software project.

## Expected Behavior

Participants should:

- Keep discussion constructive, factual, and respectful.
- Focus on coaching ideas, repository content, and public resources.
- Share only information that is appropriate for a fully public forum.
- Correct mistakes without personal attacks.
- Respect moderator requests about privacy, safety, and scope.

## Unacceptable Behavior

The following are not allowed:

- Harassment, insults, threats, or discriminatory behavior.
- Posting athlete medical details, injury histories, treatment information, or mental health details.
- Posting private contact information, school schedules, travel details, or recruiting identifiers.
- Posting private, unlisted, paywalled, or otherwise non-public video links or screenshots.
- Doxxing, gossip, or speculation about identifiable athletes, especially minors.
- Presenting individualized medical, legal, or safeguarding advice as if this repository is a professional service.
- Spam, scams, phishing, impersonation, or AI-generated misinformation presented as authoritative fact.
- Posting secrets, access tokens, credentials, or internal-only URLs.

## Scope

This Code of Conduct applies to:

- GitHub Discussions
- Issues
- Pull Requests
- Review comments
- Repository-linked community conversations run by maintainers

## Moderation

Maintainers may edit, hide, lock, or remove content that creates privacy, safety, legal, or abuse risk. Maintainers may also block repeat offenders.

Moderation priorities for this repository are:

- Protect minors and athlete privacy.
- Remove medical or recruiting disclosures.
- Remove secrets and private links quickly.
- Stop harassment and trust abuse early.

## Reporting Conduct Problems

Do not escalate conduct problems by arguing in public threads if personal information or safety concerns are involved. Flag the content through GitHub and contact the maintainers privately when possible.

## Maintainer Notes

For GitHub Discussions, maintainers should pin a safety notice that clearly says:

- Discussions are public.
- Do not post private athlete information.
- Do not post medical details.
- Do not post non-public video links.
- Do not report vulnerabilities publicly.
