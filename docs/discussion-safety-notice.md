# Pinned Discussion Safety Notice

Use this as the pinned maintainer post or category description for GitHub Discussions.

## Suggested Text

This repository's Discussions are public.

Before posting:

- Do not include athlete medical details, injury histories, treatment information, or mental health disclosures.
- Do not post private contact information, school schedules, travel details, or recruiting identifiers.
- Do not post anything about minors that is not already intentionally public.
- Do not post unlisted, private, team-only, or otherwise non-public video links or screenshots.
- Do not post access tokens, API keys, credentials, or internal URLs.

Use Discussions for public coaching questions, repository feedback, and discussion of public resources only.

Do not use Discussions to report repository security issues. Use the process in [SECURITY.md](../SECURITY.md).

Maintainers may hide or remove content that creates privacy, safety, or abuse risk.
